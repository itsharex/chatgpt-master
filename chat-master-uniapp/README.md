## 前置要求

### Node

`node` 需要 `^14 || ^18 ` ，使用 [nvm](https://github.com/nvm-sh/nvm) 可管理本地多个 `node` 版本

```shell
node -v
```

## 安装依赖

### 前端
根目录下运行以下命令
```shell
cp config-template.js config.js
npm install
```
